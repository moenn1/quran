# QuranKit API

This directory is reserved for the QuranKit FastAPI application.

## Planned Responsibilities

- Serve exact Quran text, metadata, and search endpoints with visible source attribution.
- Keep bookmarks, notes, and reading progress private by default.
- Expose semantic-search endpoints only with clear textual-similarity wording.

## Current Status

The production API is not implemented yet in this repository surface.

Use `compose.yaml`, `docker/api.Dockerfile`, and `docs/api.md` as the current bootstrap contract until the real FastAPI service lands here.
