QuranKit Bootstrap

Respectful, source-transparent Quran tooling.

This bootstrap web service exists to validate self-hosting, privacy defaults, and release documentation while the production API, CLI, and study interface are still being built.

منصة مفتوحة المصدر للدراسة والتطوير

Private by default

Bookmarks, notes, and reading progress stay private unless a future feature explicitly changes that behavior.

Semantic search guardrail

Semantic search is described as textual similarity only, never as tafsir, fatwa, or religious ruling.

Self-hosting base

Docker Compose provisions PostgreSQL, the bootstrap API, the bootstrap web service, and an optional Qdrant profile.
